## Overview
This repository demonstrates basic understandings of QGIS functionality through the completion of 10 tutorials available on [qgistutorials.com](https://www.qgistutorials.com/en/index.html). Screenshots of the conclusions of these tutorials can be found in the [outputs](outputs/) folder and the QGIS files developed from the tutorials can be found in the [projects](projects/) folder. These tutorials covered map making, projections, data importing, data visualization, raster and vector formatting, joining spatial and tabular data, preforming queries, and a variety of other GIS skills.

## Software Environment
- QGIS Version Used: QGIS Desktop 3.44.9
- Operating System: Windows 11
- Plugins used: none

## Reflection
The hands-on aspect of the completed tutorials allowed for an in-depth understanding of the layout and functionality of QGIS. By following the clear, step-by-step instructions provided it was possible to easily locate tools and observe the capabilities of the program. Due to the clarity of the instructions, the only challenge I encountered was within the Spatial Queries tutorial in step 12 when attempting to reproject the metro stations file. The Reproject Layer tool was running with no errors, but the developed file had no points within it. The bars and pubs file had no issues in the previous step and repetition with extra care to details made no difference in the problem. Researching the issue provided a fix in exporting the layer and reprojecting via the export feature to bypass the Reproject Layer tool. The fix was successful and it was possible to proceed with the rest of the tutorial. The skills learned through these tutorials will allow future QGIS projects to be completed swiftly and with a greater understanding of the capabilities of the program.
